# ARA Extended Tests

This directory contains additional experiments for the Adaptive Regime Aggregation (ARA) framework.

These tests are **separate from the reference experiments used in the Zenodo paper** and are intended for:

- stress testing the ARA architecture
- sensitivity analysis of parameters and regimes
- comparative evaluation against alternative aggregation methods
- exploratory analysis of new scenarios

The results in this directory should be interpreted as **experimental analyses** and not as part of the baseline reproducibility package for the paper.

## Structure

stress_tests/  
Extreme scenarios used to examine robustness of ARA (noise, conflicts, shocks, missing data).

sensitivity/  
Parameter sensitivity analysis (regime parameters, reliability weighting, noise levels, anchors).

comparisons/  
Comparative experiments between ARA and other aggregation approaches (e.g. mean aggregation, WSM, etc.).

results/  
Outputs generated by the extended experiments.

- `csv/` – numerical experiment results  
- `figures/` – plots and visualizations

## Purpose

The goal of this directory is to provide a **research sandbox** for extending and stress-testing ARA while keeping the official Zenodo reference implementation clean and reproducible.
