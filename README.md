# Adaptive Regime Aggregation (ARA)

Adaptive Regime Aggregation (ARA) is a formal framework for aggregating heterogeneous criteria and coordinating multiple informational subsystems under limited information.

The method is based on a strictly convex quadratic coordination functional that yields a unique global solution under explicit structural assumptions.

ARA is designed for decision systems where multiple informational regimes (e.g. data-driven and intuition-based subsystems) must be aggregated while preserving structural balance.
